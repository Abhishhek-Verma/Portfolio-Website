## 📜 How to Use
1. Clone the repository:
   ```sh
   git clone https://github.com/Abhishhek-Verma/Portfolio-Website.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Portfolio-Website
   ```
3. Open `index.html` in your browser to explore the portfolio.
